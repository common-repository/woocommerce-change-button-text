== Description ==

This WooCommerce extension gives ability to the store owner to change the 'add to cart' button text on most of the products. A great WooCommerce extension to promote catalog products or promote sales in WooCommerce.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
